cm4twc python package¶

Overview¶

The Python package cm4twc is a hydrological framework which combines interchangeable modelling components for the surface layer, subsurface, and open water parts of the terrestrial water cycle. It is designed to foster collaborations between land surface and hydrological modelling communities.
