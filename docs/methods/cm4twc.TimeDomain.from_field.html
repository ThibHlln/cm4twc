cm4twc.TimeDomain.from_field¶

classmethod TimeDomain.from_field(field)¶

Initialise a TimeDomain from a cf.Field instance.

Parameters
field: cf.Field object

The field object who will be used to initialise a TimeDomain instance. This field must feature a ‘time’ construct, and this construct must feature a ‘units’ and a ‘calendar’ property.

Examples

>>> import cf
>>> f = cf.Field()
>>> d = f.set_construct(
...     cf.DimensionCoordinate(
...         properties={'standard_name': 'time',
...                     'units': 'days since 1970-01-01',
...                     'calendar': 'gregorian',
...                     'axis': 'T'},
...         data=cf.Data([0, 1, 2, 3])
...     ),
...     axes=f.set_construct(cf.DomainAxis(size=4))
... )
>>> td = TimeDomain.from_field(f)
>>> print(td)
TimeDomain(
    time (4,): [1970-01-01 00:00:00, ..., 1970-01-04 00:00:00] gregorian
    bounds (4, 2): [[1970-01-01 00:00:00, ..., 1970-01-05 00:00:00]] gregorian
    calendar: gregorian
    units: days since 1970-01-01
    timedelta: 1 day, 0:00:00
)
