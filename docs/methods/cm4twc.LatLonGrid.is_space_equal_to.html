cm4twc.LatLonGrid.is_space_equal_to¶

LatLonGrid.is_space_equal_to(field, ignore_z=False)¶

Compare equality between the Grid and the spatial (X, Y, and Z) dimension coordinate in a cf.Field.

The coordinate values, the bounds, and the units of the field are compared against those of the Grid.

Parameters
field: cf.Field

The field that needs to be compared against TimeDomain.

ignore_z: bool, optional

Option to ignore the dimension coordinate along the Z axis. If not provided, set to default False (i.e. Z is not ignored).
