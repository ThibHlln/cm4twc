cm4twc.Model.spin_up¶

Model.spin_up(start, end, cycles=1)¶

Run model spin-up simulation to initialise model states.

Parameters
start, end: datetime object

The start and the end of the spin-up simulation. Their calendar and units are assumed to be the ones of the model Components that will be spun up. Note, the end parameter refers to the end of the last timestep in the spin-up period.

Parameter example:

start=datetime.datetime(2019, 1, 1)

Parameter example:

end=datetime.datetime(2020, 1, 1)
cycles: int, optional

The number of repetitions of the spin-up simulation. If not provided, set to default 1 (cycle).
